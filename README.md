# 📊 Madhav Ecommerce Sales Dashboard

This project is an interactive **E-commerce Sales Dashboard** built to analyze sales performance, profit trends, and customer behavior.

It helps in understanding key business metrics like revenue, quantity sold, profit, and payment methods in a simple and visual way.

---

## 🚀 Features

- 📈 Sales overview (Total Amount, Quantity, Profit, AOV)
- 🌍 State-wise sales analysis
- 🛍️ Category-wise quantity distribution
- 💳 Payment mode analysis (COD, UPI, EMI, Credit Card)
- 📅 Monthly profit trends
- 👥 Customer-wise purchase insights
- 📦 Sub-category profit comparison

## 📂 Dataset Details

The dataset includes:
- Order Amount
- Quantity
- Profit
- State
- Category & Sub-category
- Customer Name
- Payment Mode

---

## 🎯 Key Insights

- Maharashtra has the highest sales.
- Clothing category contributes the most (61%).
- COD is the most used payment method (44%).
- Tables generate the highest profit among sub-categories.
- Profit dropped in May compared to April.
